Dependencies :

You need to install it before you start creating a project. - Ruby 1.9+ | documentation - Node.js | documentation - Grunt.js | documentation - SASS | documentation - Git | documentation - Bower | documentation - COMPASS (default) | documentation - LIBSASS (optional) | documentation *** After you install all of the dependencies, you can type this on your terminal or command prompt : ```bash [sudo] gem install anila ```
It will installing Anila Command Line Interface. After that you can start using Anila for your project by using : ```bash anila new [project-folder] ``` ***

Anila CLI Commands

```bash anila version ``` Checking Anila CLI version ```bash anila new [project-folder] ``` Anila will automatically creating your project folder. By default, Anila is using Compass. ```bash anila update ``` Run it on your root project folder, it will updating Anila components using Bower. ***

Using Grunt + Libsass :

If you wish to using Libsass (yes, faster than Compass), you can run : ```bash anila new [project-folder] --libsass ```
