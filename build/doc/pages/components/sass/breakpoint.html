--- title: Breakpoint layout: build/doc/layouts/sass.html --- *** # Breakpoint

If you wondering to create your own stylesheet for every devices, we cover several breakpoints. You can use it on your media queries.

***
A default range. And you may change it as a custom value.
{{#markdown}} ```scss $sm-range: (0, 640px) !default; $md-range: (641px, 1024px) !default; $lg-range: (1025px, 1440px) !default; $xl-range: (1441px, 1920px) !default; $xxl-range: (1921px) !default; ``` {{/markdown}}
  • Small range for mobile-only.
  • Medium range for tablet-only.
  • Large range for a laptop or a standard desktop screen size.
  • Extra large for a bigger desktop screen size.
  • Extra extra large for a screen size more than an usual desktop screen size.
We also cover for landscape and potrait media queries.
{{#markdown}} ```scss $landscape: "#{$screen} and (orientation: landscape)" !default; $portrait: "#{$screen} and (orientation: portrait)" !default; ``` {{/markdown}}
***

How to use it as your media queries?

SCSS :
{{#markdown}} ```scss @media #($sm-only) { @content; } ``` {{/markdown}}
